# نظام توزيع الآيس كريم - Ice Cream Factory Distribution System

نظام إدارة توزيع الآيس كريم للمصانع مع دعم كامل للغة العربية وخرائط جوجل.

## المتطلبات

- PHP 8.0 أو أحدث
- MySQL 8.0 أو أحدث
- خادم ويب (Apache/Nginx)
- مفتاح Google Maps API

## التثبيت

### 1. قاعدة البيانات

قم بإنشاء قاعدة البيانات وتشغيل ملف SQL:

```bash
mysql -u root -p < database.sql
```

أو قم بتشغيل الملف يدوياً في phpMyAdmin أو أي أداة إدارة MySQL.

### 2. إعدادات قاعدة البيانات

قم بتعديل ملف `config.php` وأدخل بيانات قاعدة البيانات الخاصة بك:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'ice_cream_factory');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
```

### 3. Google Maps API Key

1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
2. أنشئ مشروع جديد أو اختر مشروع موجود
3. قم بتمكين APIs التالية:
   - Maps JavaScript API
   - Geocoding API
   - Directions API
4. أنشئ API Key من قسم "Credentials"
5. ضع المفتاح في ملف `config.php`:

```php
define('GOOGLE_MAPS_API_KEY', 'YOUR_API_KEY_HERE');
```

**ملاحظة:** Google Maps API لديها خطة مجانية تصل إلى $200 شهرياً، وهو كافٍ لمعظم الاستخدامات.

### 4. رفع الملفات

قم برفع جميع الملفات إلى خادم الويب الخاص بك.

## الميزات

### 1. الصفحة الرئيسية (الخريطة)
- عرض جميع العملاء على خريطة Google Maps
- مؤشرات ملونة:
  - أخضر: عملاء لديهم طلبات اليوم
  - رمادي: عملاء بدون طلبات
  - أحمر: موقع المصنع
- فلاتر لعرض جميع العملاء أو طلبات اليوم فقط

### 2. إدارة العملاء
- إضافة عملاء جدد مع العناوين
- تحديد الموقع تلقائياً من العنوان
- عرض قائمة جميع العملاء
- حذف العملاء

### 3. إدارة السائقين
- إضافة سائقين جدد
- تحديد قدرة كل سائق (عدد الطلبات في اليوم)
- تفعيل/تعطيل السائقين
- حذف السائقين

### 4. الطلبات اليومية
- إنشاء طلبات جديدة لليوم
- اختيار العملاء الذين لديهم طلبات
- تعيين السائقين للطلبات
- عرض المسارات المحسّنة على الخريطة
- توزيع الطلبات على السائقين

### 5. موقع المصنع
- إضافة موقع المصنع (مرة واحدة)
- تحديد الموقع على الخريطة
- استخدام الموقع كنقطة بداية للمسارات

## الاستخدام

1. **ابدأ بإضافة موقع المصنع** من صفحة "موقع المصنع"
2. **أضف العملاء** من صفحة "العملاء"
3. **أضف السائقين** من صفحة "السائقين"
4. **أنشئ الطلبات اليومية** من صفحة "الطلبات اليومية"
5. **قم بتعيين السائقين** للطلبات
6. **اعرض المسارات** على الخريطة

## الأمان

- لا يوجد نظام مصادقة حالياً (كما طلبت)
- في حالة النشر على الإنترنت، يُنصح بإضافة نظام مصادقة
- تأكد من حماية ملف `config.php` من الوصول المباشر
